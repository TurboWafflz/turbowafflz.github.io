Lintest

Lintest is a simple program for showing information about a Linux computer.

To install dependencies on Ubuntu, run: ./install-depends-ubuntu.sh in a terminal in the directory you extracted the zip to.
Not an Ubuntu user? No problem, just manually install the following packages:
boxes
figlet
toilet
ncurses-bin
speedtest-cli
lolcat
cmatrix

To run, type: ./lintest.sh into a terminal in the directory you extracted the zip to.
