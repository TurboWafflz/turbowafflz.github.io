KeySpeed

KeySpeed is a set of tools for testing how fast you can keymash.
  • MashSpeed: Pure keymashing, any keypress is counted.
  • ShiftSpeed: Similar to SingleMashSpeed, but both left and right shift are counted.
  • SingleMashSpeed: Only one key is counted, this is automatically set when you start.
All programs require one argument, the number of seconds to test for.

Example:
To run MashSpeed with a 10 second time limit, type: python3 mashspeed.py 10 into a terminal in the same folder as the script.
